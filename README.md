# How to run App

## Open your Terminal:
```
    git clone https://github.com/LuisCamargoM/antRacing.git
```

## In the root folder:
```
    1 - yarn or npm install
    2 - cd ios && pod install && cd ..
    3 - yarn ios (iOS Simulator) or yarn android (Android Emulator)
```

#### Obs: Ruby Version: 2.7.5
